方法一

下载并执行NodeSource安装脚本

curl -sL https://deb.nodesource.com/setup_18.x | sudo -E bash -

这个脚本将会添加 NodeSource 的签名 key 到你的系统，创建一个 apt 源文件，安装必备的软件包，并且刷新 apt 缓存

如果你需要另外的 Node.js 版本，例如12.x，将setup_18.x修改为setup_12.x

sudo apt install nodejs

nodejs软件包同时包含node和npm二进制包

方法二

安装n管理器来管理nodejs版本

sudo apt install nodejs
sudo npm install -g n --force
sudo n stable # 安装最新版的nodejs(stable版本)
sudo n 5.1.0  # 安装指定版本的nodejs(5.1.0)
