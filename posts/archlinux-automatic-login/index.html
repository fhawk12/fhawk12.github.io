Archlinux Automatic Login

2023-09-18

In most cases, you want to set up automatic login on a virtual console, (whose device name is ttyN, where N is a number). The configuration of automatic login for serial consoles will be slightly different. Device names of the serial consoles look like ttySN, where N is a number.

Execute systemctl edit getty@tty1 or vim /etc/systemd/system/getty.target.wants/getty@tty1.service with the following content:

/etc/systemd/system/getty@tty1.service.d/autologin.conf

[Service]
ExecStart=-/sbin/agetty -o '-p -f -- \\u' --noclear --autologin username %I $TERM

Tips: Don't reboot immediately, use clt+alt+F2 switch to tty2 and see whether it is successful or not
