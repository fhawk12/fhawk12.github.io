Archlinux Automatic Login

2025-01-04fhawk12
#linux 

Execute command systemctl edit getty@tty1 or vim /etc/systemd/system/getty.target.wants/getty@tty1.service, and modify some configuration

[Service]
ExecStart=-/sbin/agetty -o '-p -f -- \\u' --noclear --autologin username %I $TERM

tips: Don’t reboot immediately, press ctrl+alt+F2 to try if it works
