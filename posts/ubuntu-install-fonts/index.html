Ubuntu Install Fonts

1.安装mkfontscale,mkfontdir,fc-cache命令

如果运行mkfontscale命令时提示mkfontscale: command not found,则需要首先安装以下命令，安装方法如下：

#使mkfontscale 和 mkfontdir命令正常运行
sudo apt-get install ttf-mscorefonts-installer
#使fc-cache命令正常运行
sudo apt-get install fontconfig

2.下载字体

如果系统中没有中文字体，需先安装中文字体

cd /usr/share/fonts/

如果目录不存在就自己创建

3.刷新系统字体缓存

cd /usr/share/fonts/
sudo mkfontscale  #生成核心字体信息
sudo mkfontdir    #生成字体文件夹
sudo fc-cache -fv #刷新系统字体缓存
