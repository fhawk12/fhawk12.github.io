Git Cannot Push

2024-01-21

when you execute the command git push, maybe will show the message

ssh: connect to host github.com port 22: Connection timed out fatal: Could not from remote repository.

Please make sure you have the correct access rights

and the repository exists.

add file to ~/.ssh/config and edit it to solve it

Host github.com
    Hostname ssh.github.com
    Port 443
    User git
