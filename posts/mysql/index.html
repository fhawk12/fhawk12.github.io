#mysql 

Common commands⌗

  • Add user: create user 'username'@'localhost' identified by 'password';
  • Query users: select User from mysql.user;
  • Grant user: grant all privileges on databasename.tablename to 'username'@'localhost';
  • Query privileges: show grants for 'username';
  • Change password: set password for username@localhost = password(‘password’);
  • Delete user: drop user username;
